Hand Detection

“Design and implement a real-time hand gesture–based control system that enables users to adjust system volume and screen brightness without physical input devices, providing a natural, touchless, and accessible human–computer interaction solution.”

Problem Statement

Traditional volume and brightness controls require physical interaction with hardware or peripherals, which can be inconvenient, unhygienic, or inaccessible in certain scenarios. There is a need for a touchless, intuitive system that allows seamless system control using natural hand gestures in real time.

Literature Review / Market Research

Prior research using OpenCV and Google MediaPipe Hands demonstrates accurate real-time hand landmark detection using standard webcams, enabling gesture-based interaction without specialized hardware.

Research Gap / Innovation

Existing systems rely on static gestures and discrete actions. This project introduces continuous, smooth gesture-to-control mapping using finger-distance metrics for precise real-time volume and brightness adjustment.

System Methodology

Dataset / Input

Real-time webcam input using pretrained Google MediaPipe hand landmark models. No custom dataset required.

Model / Architecture

Two-stage pipeline: BlazePalm for detection → Hand Landmark Model (21 points) → distance mapping → system control APIs.

Live Execution

No Live Demo

Results & Analysis

Effective Accuracy 96%

The system achieves smoother and more precise control than existing approaches by using continuous finger-distance mapping and temporal smoothing, significantly reducing jitter and unintended adjustments.
